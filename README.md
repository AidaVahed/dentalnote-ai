# dentalnote-ai

Eine einfache Patientenverwaltungs-App mit Flask-Backend und Streamlit-Frontend.  
Inklusive PDF-Upload zur automatischen Generierung medizinischer Observations-Texte über OpenAI.

---

## Features

- Patienten anlegen, auflisten und bearbeiten  
- PDF-Transkripte hochladen und automatisch Observationen generieren  
- Einfache Bedienung über Streamlit-Weboberfläche
